As a result of plotting histogram of number of missing or unknown values in each column, following 6 columns had more than 200,000 of missing values.

  1. AGER_TYP; missing 685,843 values
    This column shows best ager typology, which is a term in marketing and social science research to describe target groups of people over 50 years old. One of the reasons why this column has many missing values is because those people under age of 50 are classified into '-1: unknown' or '0: no classification possible' and they are changed to NaN in the previous step.
  1. GEBURTSJAHR; missing 392,318 values
    This is year of birth data.
  1. TITEL_KZ; missing 889,061 values
    This column is used to fill an academic title as 'Dr.' and 'Prof.' if a person has one. Since a person who has an academic title is limited, it is acceptable that this column has many unknown fields.
  1. ALTER_HH, missing 310,267 values
    This column shows birthdate data of head of household.
  1. KK_KUNDENTYP, missing 584,612 values
    KK_KUNDENTYP is consumer pattern over past 12 months. In this column if a person is not a customer, data is classified as unknown.
  1. KBA05_BAUMAX, missing 476,524 values
    KBA05_BAUMAX is a most common building type within the microcell.

For this analysis, we have removed all the features with more than 200,000 missing or unknown data to use more informative features.

Step 2.3: Interpret Principal Components¶

Now that we have our transformed principal components, it's a nice idea to check out the weight of each variable on the first few components to see if they can be interpreted in some fashion.

As a reminder, each principal component is a unit vector that points in the direction of highest variance (after accounting for the variance captured by earlier principal components). The further a weight is from zero, the more the principal component is in the direction of the corresponding feature. If two features have large weights of the same sign (both positive or both negative), then increases in one tend expect to be associated with increases in the other. To contrast, features with different signs can be expected to show a negative correlation: increases in one variable should result in a decrease in the other.
